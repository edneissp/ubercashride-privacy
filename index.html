Política de Privacidade - UberCashRide

O aplicativo UberCashRide respeita a privacidade de seus usuários. As informações coletadas são utilizadas exclusivamente para autenticação via API da Uber e armazenamento seguro de dados estatísticos de corridas no Supabase.

1. Coleta de dados

Coletamos apenas informações autorizadas pelo usuário durante o processo de login com a conta Uber.

2. Uso de dados

Os dados são usados para análise de desempenho, tempo e ganhos por corrida. Nenhum dado é compartilhado com terceiros.

3. Armazenamento

As informações são armazenadas de forma segura no Supabase, em conformidade com a LGPD e GDPR.

4. Direitos do usuário

O usuário pode solicitar a exclusão de suas informações a qualquer momento, entrando em contato pelo suporte do aplicativo.

5. Segurança

O UberCashRide utiliza autenticação segura e criptografia padrão de mercado para proteger todos os dados.
